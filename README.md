## 🔧 Setup & Installation

1. Clone the repository:
```bash
git clone https://github.com/amoesiemmanuel/portfolio.git
```

2. Navigate to the project:
```bash
cd portfolio/Public
```

3. Open in browser or run with a local server:
```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx http-server
```

4. Visit `http://localhost:8000` in your browser
